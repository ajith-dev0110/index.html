# 🛍️ Tech&Threads - React Product Listing App

A dynamic, responsive product listing application built with React. This project demonstrates core React concepts including functional components, state management (Hooks), prop drilling, event handling, and conditional rendering.

## ✨ Features
* **Dynamic Product Grid:** Displays products with details (image, name, price, category, description).
* **Real-time Search:** Filter products instantly by name (case-insensitive).
* **Category Filtering:** View products by specific categories (Electronics, Clothing, Accessories).
* **Smart Sorting:** Sort items by Price (Low-High/High-Low) and Name (A-Z/Z-A).
* **Shopping Cart functionality:** Add items, view total price, and remove items.
* **Responsive Design:** Seamless layout transitions from mobile to desktop.

## 🛠️ Technology Stack
* React 18 (Functional Components & Hooks)
* Vite (Build Tool)
* Pure CSS (Responsive Grid & Flexbox)
